Aula 01 – Repetições (Parte 1)

While

A declaração while cria um laço que executa uma rotina especifica enquanto a condição de teste for avaliada como verdadeira. A estrutura while tem o teste lógico no começo do bloco, antes mesmo de executar-se as instruções testa se a condição e true, isso é muito importante, pois, temos uma estrutura muito parecida mais que faz o teste lógico no final da execução do bloco, ou seja, primeiro se executa a instrução depois verifica-se se a condição é verdadeira para fazer o laço de repetição entrar em looping.


Sintaxe:


                while (condição) {
                    rotina
                }

Condição


É uma expressão avaliada anetes de cada passagem através do laço. Se essa condição for avaliada como verdadeira, a rotina é executada. Quando a condição for avaliada como falsa, o programa segue o fluxo passando para a próxima linha de comando logo abaixo do fechamento de chaves do bloco while.


Rotina


Rotina é uma declaração que é executada enquanto uma condição é avaliada como verdadeira. Para executar múltiplas declarações devemos usar as chaves ({…}) para agrupá-las.


Exemplo:


                var controle = 0
                var n1 = 0

                while (controle < 3) {
                    controle++
                    n1 = n1 + controle
                    // n1 += controle
                }

Do... while ()

De forma muito parecida com o while existe a estrutura do… while () {}, que realiza o teste lógico condicional no final do bloco de comandos, ela é muito útil para casos em que precisamos executar o código antes de testar a condição.


Exemplo:


                var controle = 0

                do {
                    instrução
                    controle++ ou controle += 1 ou controle = controle + 1
                } while (controle >= 10)
