Arrays

Arrays são objetos semelhantes a listas que vêm com uma série de métodos embutidos para realizar operações de travessia e mutação. Nem o tamanho de um array JavaScript nem os tipos de elementos são fixos, já que o tamanho de um array pode ser alterado a qualquer momento e os dados podem ser armazenados em posições contíguas, arrays JavaScript não tem a garantia de serem densos, isso depende de como o programador escolhe usá-los, se esses recursos não são desejáveis para o seu caso em particular, você pode considerar usar arrays tipados.


Arrays não podem usar strings como índices, devem ser usados números inteiros.


Sintaxe


var frutas = [] // Nesse caso declaramos um vetor vazio que será preenchido no decorrer do código.

var frutas = [Maçã, Banana] // Declaração do vetor com dois itens dentro dele.

frutas.push('Laranja') // Adicionar um item ao final do array.

frutas.pop('') // Remover um item do final do array.

frutas.shift('') // Remover um item do início da lista.

frutas.unshift('Morango') // Adiciona um item ao início do array.

frutas.indexOf('Banana') // Procura o índice de um item dentro do array.
