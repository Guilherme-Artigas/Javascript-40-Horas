Condições aninhadas e a estrutura SWITCH

Continuando com nossos estudos, é rotina dentro da programação encontrarmos situações de que precisaremos de um condigo mais complexo quando o assunto for condicionais, pois, existem diversas situações que iram precisar testar múltiplos caminhos/opções, nesses casos trabalharemos com estruturas condicionais aninhadas e a estrutura SWITCH.


Veja um exemplo da sua sintaxe:


                if (condição1) {
                    instrução1
                } else if (condição2) {
                    instrução2
                } eles if (condição3) {
                    instrução3
                }
