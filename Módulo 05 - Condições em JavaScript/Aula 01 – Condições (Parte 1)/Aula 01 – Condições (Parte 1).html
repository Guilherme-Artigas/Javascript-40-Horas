Condições (Parte 1)

Desvio condicional

A condicional if é uma estrutura que executa a afirmação, dentro do bloco, se determinada condição for verdadeira. Se for falsa, executa as afirmações dentro de else.


Condição


Dentro da programação a condicional é uma expressão (premissa) que pode ser avaliada como verdadeira (true) ou falsa (false), é necessário ter conhecimentos básicos de lógica de programação para entender esse conceito de uma melhor forma.


Sintaxe:


                if (condição) {
                    instrução1
                }
                else {
                    instrução2
                }
