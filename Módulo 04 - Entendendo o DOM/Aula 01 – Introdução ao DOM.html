Introdução ao DOM

Entendendo o DOM

O modelo de objeto de documentos (do inglês Document Object Model, DOM) é uma API definida pelo W3C para representar e interagir com qualquer documento HTML e XML.


O DOM é um modelo de documento carregado pelo navegador. Este documento é representado através de uma árvore de nós, onde cada um destes nós representa uma parte do documento (por exemplo: elemento, texto ou comentário).


O DOM é uma das APIs mais usadas da web, por que ele permite que cada codigo rodando no navegador acesse e interaja com cada nó do documento. Os nós podem ser criados, movidos ou modificados.


NOTA: Tudo dentro do JavaScript está dentro de um objeto chamado window, dentro do window temos vários outros objetos podemos citar como exemplo location, document, history, seguindo a construção da nossa árvore hierárquica temos como filho de document o html e filhos de html são head e o body, todo site possui uma árvore hierárquica contendo pais e filhos.



Formas de acesso aos elementos com JS


Por Marca (Tag): window.document.getElementsByTagName()

Por ID: window.document.getElementById()

Por Nome: window.document.getElementsByName()

Por Classe: window.document.getElementsByClassName()

Por Seletor (presente nas versões mais recentes do EcmaScript): window.document.querySelector()
