Armazenando Dados

Aula explicando os conceitos iniciais de armazenamento de dados na programação, com a linguagem JavaScript. No JavaScript moderno, além de utilizar a palavra var, também podemos usar a palavra let.


Damos o nome para cada variável declarada de identificador, existem regras para a declaração de variáveis:


  • Podem começar com letra, $ ou _
  • Não pode começar com números
  • É possível usar letras e números de forma mesclada
  • É possível usar acentos e símbolos
  • Não pode conter espaços
  • Não podem ser palavras reservadas

Dicas para a declaração de variáveis.


  • Maiúsculas e minúsculas fazem diferença
  • Tente usar nomes coerentes para as variáveis
  • Evite se tornar um “programador alfabeto” ou um “programador contador”
