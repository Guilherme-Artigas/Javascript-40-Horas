Tratamento de dados

Number e String

Aula falando sobre coleta de dados e a importância do tratamento dessas informações dentro da programação.


Aprendemos nessa aula que o comando window.prompt(‘’) usado para capturar um dado do usuário sempre ira retornar valores em string (texto), isso é uma característica do comando, nesse caso precisamos falar sobre a conversão de tipos de dados para o tratamento correto deles.


Conversão para números inteiros usamos:

Number.parseInt(nome da variável)


Conversão para números com ponto flutuante usamos:

Number.parseFloat(nome da variável)


Nas versões mais recentes da linguagem JavaScript podemos definir somente como Number e a própria linguagem vai saber definir se a número digitado é inteiro ou real (número com ponto flutuante).


Podemos chamar a declaração da seguinte forma:

var numero1 = Number(window.prompt(‘Digite o 1º número’))


Conversão de número para String


String(nome da variável)

ou

nome da variável.toString()


Formatando Strings


nome da variável.lenght (Quantos caracteres a string possue)

nome da variável.toUpperCase() (Converte todas as letras para MAIÚSCULAS)

noem da variável.toLowerCase() (Converte todas as letras para minúsculas)


Formatando Números


nome da variável.toFixed(2) (Quantidade de casas decimais)

nome da variável.toLocaleString('pt-BR', {style: 'currency', currency: 'BRL'})
