Operadores (Parte 2)

Operadores que vamos ver nessa aula são os relacionais, lógicos e ternário.


Operadores relacionais


Começando com os relacionais, operadores relacionais sempre retornam o valor booleano, verdadeiro ou falso.


> Maior que

< Menor que

>= Maior ou igual a

<= Menor ou igual a

== igual a

!= diferente


Exemplo:


5 > 2 → true

7 < 4 → false

8 >= 8 → true

9 <= 7 → false

5 == 5 → true

4 != 4 → false


Operadores de identidade


== Igual

Com esse operador verificamos se o valor é igual, não se importando se é do mesmo tipo. Exemplo:

5 == ‘5’ → true


=== Idêntico

Com esse operador verificamos se além do valor é do mesmo tipo, de número para número, String para String e assim por diante. Exemplo:

5 === ‘5’ → false



Operadores lógicos


! Negação

&& Conjunção (and/e)

|| Disjunção (or/ou)


Exemplos:


1º Exemplo negação:

“Eu quero uma caneta emprestada, mais ela não pode ser azul”

ou seja, pode ser de qualquer cor que você ira satisfazer a necessidade da situação.


2º Exemplo Conjunção:

“Eu quero 2 canetas emprestadas, uma preta e uma vermelha”

ou seja, nesse caso tem que ser uma caneta preta e uma vermelha, caso contrário a pessoa não ficara satisfeita.


3º Exemplo Disjunção:

“Eu quero uma caneta emprestada, pode ser azul ou preta”

ou seja, nesse caso a pessoa está dando 2 opções para satisfazer a sua necessidade.


NOTA 1: Quando temos na mesma expressão os valores aritméticos, relacionais e lógicos, eles serão resolvidos por ordem sendo ela a seguinte: Aritméticos, Relacionais e Lógicos.


NOTA 2: Existe uma ordem de precedência para os operadores lógicos, caso os três apareçam na mesma expressão que é a seguinte: Negação, Conjunção (e), Disjunção (ou).



Operador Ternário


O operador ternário é composto por dois símbolos que aparecem na mesma expressão. A interrogação e o dois pontos: ? :


Ele se chama ternário por conter 3 partes, a primeira é antes do ponto de interrogação a segunda parte e entre o ponto de interrogação e o dois pontos e a terceira parte e depois do dois pontos.


Exemplo:

teste ? true : false


Como podemos comprovar no exemplo acima, o operador leva esse nome porque ele “trabalha” com três operandos…


Primeiro operando: É um teste lógico que retorna ou true (verdadeiro) ou false (falso).

Segundo operando: É o resultado verdadeiro da expressão lógica da primeira parte.

Terceiro operando: É o resultado falso da expressão lógica da primeira parte.


Olhando para um exemplo mais pratico, fica da seguinte forma:


media >= 7.0 ? “APROVADO” : “REPROVADO”


Nessa expressão estamos dizendo que o aluno estará aprovado caso a média das notas dele seja maior ou igual a 7.0, caso seja menor a mensagem mostrada será de reprovado.
